# FlyRank Capstone

Capstone repository for the **FlyRank AI Internship** (Week 1 — Setup).

## Purpose

This repo is the home for my FlyRank internship capstone. Week 1 establishes a working AI-assisted toolchain (Node.js LTS, Git, Cursor) and a reviewable GitHub project with conventions documented for AI assistants.

## Status

Phase 1 complete: local toolchain, GitHub repo scaffolding, and Conventional Commits history.

## Stack

- **Runtime:** Node.js LTS
- **Editor:** Cursor
- **VCS:** Git + GitHub
- **Conventions:** See [`CLAUDE.md`](./CLAUDE.md)

## Getting started

1. Install [Node.js LTS](https://nodejs.org/) and [Git](https://git-scm.com/)
2. Clone this repository
3. Open the project in Cursor
4. Read `CLAUDE.md` before asking the AI assistant to scaffold features

Verify the toolchain:

```bash
node -v
git --version
```
